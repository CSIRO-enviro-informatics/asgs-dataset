ASGS 2016 - as a Register of Registers

About

This is an API that delivers contents of the Australian Statistical Geography Standard (ASGS) 2016 product online as Linked Data. To do this, the tool uses the ASGS Ontology as its main data model and the series of public ASGS 2016 Web Feature Services, such as the one for Mesh Blocks as its source of data.

This tool is currently still in beta testing mode and, while it is, it is operated by CSIRO's Environmental Informatics team. In the medium term, the tool is likely to be managed by the Australian Bureau of Statistics.

Here you see a 'Register of Registers', each of which contains classes of item from the Linked Data representation of ASGS-2016 content.

Register

  • Meshblock Register - the list of all Meshblocks, paged
