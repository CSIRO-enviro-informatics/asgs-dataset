ASGS 2016 Linked Data API

This is an API that delivers contents of the Australian Statistical Geography Standard (ASGS) 2016 product online as Linked Data. To do this, the tool uses the ASGS Ontology as its main data model and the series of public ASGS 2016 Web Feature Services, such as the one for Mesh Blocks as its source of data.

This tool is currently still in beta testing mode and, while it is, it is operated by CSIRO's Environmental Informatics team. In the medium term, the tool is likely to be managed by the Australian Bureau of Statistics.

Other ABS Features in ASGS

  • Significant Urban Areas - the list of all SUAs, paged
    • A SUA - example, 1018 (Lithgow)
  • Greater Capital City Statistical Areas - the list of all GCCSAs, paged
    • A GCCSA - example, 3GBRI (Greater Brisbane)
  • Remoteness Areas - the list of all RAs, paged
    • A RA - example, 71 (Outer Regional Australia - NT)

Indigenous Locality Features

  • Indigenous Location Register - the list of all ILOCs, paged
    • An ILOC - example, 50800204 - Mindi Rardi - Kurnangki (Fitzroy Crossing)
  • Indigenous Area Register - the list of all IAREs, paged
    • An IARE - example, 107002 (Baulkham Hills)
  • Indigenous Area Register - the list of all IREGs, paged
    • An IREG - example, 501 (Broome)

State Section Features

  • Urban Centre and Locality Register - the list of all UCLs, paged
    • A UCL - example, 112001 (Armidale)
  • Sections of State Ranges - the list of all SOSRs, paged
    • A SOSR - example, 203 (Geelong)
  • Sections of State - the list of all SOSs, paged
    • A SOS - example, 50 (Perth)
