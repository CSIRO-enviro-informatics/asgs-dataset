ASGS 2011 Linked Data API

This is an API that delivers contents of the Australian Statistical Geography Standard (ASGS) 2011 product online as Linked Data. To do this, the tool uses the ASGS Ontology as its main data model and the series of public ASGS 2011 Web Feature Services, such as the one for Mesh Blocks as its source of data.

This tool is currently still in beta testing mode and, while it is, it is operated by CSIRO's Environmental Informatics team. In the medium term, the tool is liekly to be managed by the Australian Bureau of Statistics.

Code

The code for this application is online at https://github.com/CSIRO-enviro-informatics/asgs-dataset. The 2011 version of this product is found in a brand named asgs2011
